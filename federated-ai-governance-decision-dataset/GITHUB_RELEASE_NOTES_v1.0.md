# v1.0 Release Notes

Initial release of the Federated AI Governance Decision Dataset (FAGDD).

FAGDD is a synthetic, research-grade dataset designed to support research on federated AI governance across multi-cloud government systems. The dataset contains 500 structured decision records spanning Azure, AWS, on-premises, hybrid, and multi-cloud environments, with coverage across healthcare, justice, finance, education, smart cities, public safety, and defense.

Each record captures governance and compliance decision factors including policy framework alignment, identity context, security controls, audit trace text, risk level, compliance status, human override requirements, and final outcome.

This release includes:

- `data/fagdd_v1.csv`
- `data/fagdd_v1.json`
- deterministic generation script
- validation script
- dataset card
- citation metadata
- versioning and changelog files

The dataset is fully synthetic and does not contain real government cloud data, citizen data, classified information, or proprietary operational logs.

# Federated AI Governance Decision Dataset (FAGDD)

The Federated AI Governance Decision Dataset (FAGDD) is a synthetic, research-grade dataset created to support research on federated AI governance in multi-cloud government systems.

The dataset contains 500 structured synthetic records representing governance decisions across Azure, AWS, on-premises, hybrid, and multi-cloud environments. Records are designed to reflect realistic public-sector decision contexts involving sovereignty constraints, standards-aligned policy review, identity-aware access, security controls, auditability, and human oversight.

Each record includes:

- cloud environment
- deployment model
- government domain
- data classification
- jurisdiction
- policy framework
- governance rule
- AI action
- decision explanation
- risk level
- compliance status
- human override requirement
- final outcome
- identity context
- security control
- audit trace
- timestamp
- agency type
- region

FAGDD is intended for academic research, benchmarking, teaching, and experimental evaluation of AI governance controls in federated public-sector environments.

This dataset is fully synthetic and does not contain real government cloud logs, tenant metadata, citizen records, classified information, or proprietary operational data.

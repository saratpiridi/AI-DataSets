# Federated AI Governance Decision Dataset (FAGDD)

The Federated AI Governance Decision Dataset (FAGDD) is a synthetic, research-grade dataset for studying governance and compliance decisions in multi-cloud government AI systems.

It contains 500 realistic synthetic records covering:

- Azure
- AWS
- On-Prem
- Hybrid
- Multi-Cloud

The dataset spans public-sector domains including healthcare, justice, finance, education, smart cities, public safety, and defense.

Each record includes governance-relevant attributes such as:

- policy framework alignment
- governance rule
- AI action
- decision explanation
- risk level
- compliance status
- human override requirement
- identity context
- security control
- audit trace

FAGDD is designed for research on:

- federated AI governance
- sovereignty-aware access and policy enforcement
- explainable compliance decisions
- identity-aware controls in multi-cloud systems
- auditability and oversight in public-sector AI

This dataset is fully synthetic and safe for open publication. It does not contain real government records, classified information, proprietary cloud telemetry, or personal data.
